# Slot Machine Game (Python)

A console-based Python slot machine simulation featuring betting, spinning, and winnings calculation.

## How It Works

- Allows users to deposit money and place bets on 1-3 lines.
- Spins a 3x3 slot machine grid of symbols with different values.
- Winnings are determined based on matching lines and symbol values.
- Keeps track of balance and play session with clear CLI prompts.

## How to Run

1. Ensure you have Python 3 installed.
2. Clone or download this repository.
3. Open a terminal in the project folder.
4. Run the program 
5. Follow the prompts to deposit money, place bets, and spin the slots.
